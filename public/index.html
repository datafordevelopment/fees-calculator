FeeCalc :: About

My name is Michael. Since I started working at Dwolla it became clear to me that payment fees are really confusing. I put this calculator together so people could easily understand how different payment system fees compare.

Payment gateways have a lot of different pricing models, which means that the numbers could vary a little bit here and there. Bottom line: this tool is not 100% accurate, since you might qualify for some different pricing structure.

The source code is freely available on Github. If you have any suggestions please reach out on Twitter - I'll be happy to update this.
